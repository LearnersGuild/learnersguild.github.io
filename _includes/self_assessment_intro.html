Before you apply to join our learning collective, ask yourself: "am I prepared for the technical challenge?" Use this self-assessment to help find out

To be ready for the program, you'll need to gain some basic proficiency in two key skills: computational thinking and web development.

Use the exercises below to help you determine if you are ready and to discover resources that can help you prepare.

Once you've finished, you will know if you're ready to submit your application to Learners Guild.

If you are not quite ready, we've listed some freely available resources that you can use to learn. We also recommend seeking out help from other people; software development is a team sport, after all. One place to start is to search for "beginner programmer" or "web development" Meetups in your area.
