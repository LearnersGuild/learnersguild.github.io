Once you are sure that you are willing to commit to joining our learning collective to pursue a career in web development, the next question you should ask yourself is: "am I prepared?"

To be ready, you'll need to gain some basic proficiency in two key skills: computational thinking and web development.

Use the exercises below to help you determine if you are ready and to discover resources that can help you prepare.

Reasons you may need to prepare
To build the basic skills necessary to fully engage with our enrollment process and to start the program itself
To spend a suffient amount of time with programming to know that you like it, and know that you’re ready to commit to a career as a software developer

Basics of web development

The basics of web development covers topics like HTML, CSS, and JavaScript, as well as some conceptual understanding of how browsers and the web work.

To self-assess your web development skills, try to complete this coding challenge to make a color picker with HTML, CSS, and JavaScript. If you can complete steps 2–4 of this challenge in less than an hour with no outside help (except for Google, of course), you’re likely skilled enough in basic web development to join Learners Guild.

If you aren’t able to complete the challenge, or if it was very difficult for you, then you will want to strengthen your web development skills.

Computational thinking

Computational thinking is the range of cognitive abilities needed to program a computer. It includes abilities like logical reasoning, algorithm design, pattern recognition, and evaluating a series of sequential instructions.

To self-assess your computational thinking skills, complete the Computer Programming Aptitude Test from the University of Kent. If your score is higher than 19, you’re definitely ready for Learners Guild.

If your score is lower, you will want to practice more to build up your computational thinking ability. Luckily, computational thinking is a skill just like basketball or dance: the more you practice, the better you’ll get.

*Note: don’t limit yourself to just the above resources! There are a lot more out there if you do some searching.
